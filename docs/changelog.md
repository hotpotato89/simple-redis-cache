# Changelog

Логи версий ведутся начиная с `1.1.0`, как со стабильного релиза.

## [1.1.0] - 2026-07-13

### Added
- Добавлена поддержка `pickle`-сериализации через параметр `use_pickle=True` в декораторе `@cache`
- Добавлен маркер `PICKLE:` для автоматического определения формата при чтении из Redis
- Добавлена обработка `bytes` и `str` при чтении из Redis
- Добавлены тесты для `pickle`-режима

### Fixed
- Исправлен баг с потерей типа объекта при чтении из кэша

### Known Issues
- **Не используйте `decode_responses=True`** в Redis-клиенте при включённом `use_pickle=True`
